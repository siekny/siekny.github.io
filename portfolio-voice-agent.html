AI Voice Agent

Developing an AI voice assistant for a Poomsae training kiosk.

Introduction

Developed an embedded AI voice assistant for a poomsae training kiosk that allows users to navigate the application, control practice sessions, and ask coaching questions entirely by voice — running on a Windows kiosk with internet dependency.

  • Designed a 5-stage voice pipeline (IDLE → WAKEWORD → LISTENING → THINKING → SPEAKING) as a background daemon thread, exposing a single event callback to the UI layer
  • Every component (STT, LLM, TTS) has a local and cloud implementation behind a unified interface
  • Integrated the agent into a PySide6/QML application - no cloud wake word service needed

Wake Word & Speech

  • Built a custom on-device wake word detector
  • Implemented barge-in interruption

LLM & RAG

  • Designed a task classifier that routes user speech into intents before dispatching to the appropriate agent
  • Built a RAG coaching QA system
  • Built a RecommendationEngine that queries the user's local DB training history on login and uses the LLM to generate a personalized next-step greeting.
